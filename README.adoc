== Descripción

Sistema de monitoreo remoto para el inversor fotovoltaico **Riello H.P.6065REL-D** (6 kW, 3 MPPTs), que reemplaza el software propietario SunVision (inestable, solo Windows) por una solución basada en software libre, accesible desde cualquier navegador web.

El protocolo de comunicación **SISER (Phoenixtec)** fue descubierto mediante decompilación de SunVision (clase `SISERBus.java`), reemplazando la hipótesis inicial de Modbus RTU. El daemon `siser-reader` (Python) se comunica con el inversor vía RS232 (adaptador CH340 USB-RS232), realizando un handshake de registro (`offlineEnquiry` + `sendAddress`) y control de flujo RTS/DTR para el optoacoplador interno.

== Protocolo SISER (Phoenixtec)

El inversor H.P.6065REL-D **no usa Modbus RTU**. Usa el protocolo binario propietario SISER (Phoenixtec), descubierto por decompilación de SunVision.

=== Handshake de Registro

. `offlineEnquiry`: El daemon envía una trama de registro al inversor
. `sendAddress`: El inversor responde asignando una dirección (típicamente 33)
. `readMichele`: Lectura de datos con control RTS/DTR para el optoacoplador

== Base de Datos (TimescaleDB)

[cols="1,1,1,1,1",options="header"]
|===
| Tabla | Intervalo | Origen | Estado | Propósito

| realtime | ~5 seg | siser-reader | *26K+ filas* | Dashboard live
| fast_samples | 1 min | siser-reader | *VACÍA* | Gráficos hora/día
| slow_samples | 15 min | continuous aggregate | *VACÍA* | Tendencias
| hourly_energy | 1 hora | continuous aggregate | *VACÍA* | Tendencias
| daily_energy | 1 día | continuous aggregate | *VACÍA* | Tendencias
| cumulatives | 1 min | siser-reader | *VACÍA* | Energía total, horas
| daily_production | 1x/día | siser-reader | *VACÍA* | 48 slots horarios
| events | evento | siser-reader | *VACÍA* | Alarmas
|===

NOTE: Las tablas `fast_samples`, `cumulatives` y `daily_production` están vacías porque siser-reader aún no escribe en ellas. Solo escribe en `realtime`. Las continuous aggregates dependen de `fast_samples`.

== Inicio Rápido

. Clonar el repositorio:
+
----
git clone https://github.com/drumino/medidor-inversor.git
cd medidor-inversor/Proyecto
----

. Copiar `.env` template y configurar passwords:
+
----
cp .env.example .env
# Editar .env con passwords seguros
----

. Levantar servicios:
+
----
docker compose up -d
----

. Verificar:
+
----
docker compose ps
docker compose logs -f siser-reader
----
